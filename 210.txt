      pwd:
      git init:初始化一个仓库
      touch index.html:创建一个名为index.html的文件
      git status:查看当前状态
      git add index.html:添加一个名为index.html的文件到缓存区
      git commit -m '第一次上传'：声明
      git remote add https://Chentongyong/jds.git
      （说明上传路径：包括用户名、库名）
      git remote add origin https://github.com/Chentongyong/jds.git
      (说明上传的分支（这里上传的是主分支）)
      git push origin master (在push与origin之间可添加-u（可写可不写）添加到主分支)
      git pull --rebase origin master:将github上jds库里的README.md同步到本地的jds文件夹中
